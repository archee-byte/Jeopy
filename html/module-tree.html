Module Hierarchy

  • AdminGui: AdminGui is the administrative interface's main class.
  • ButtonGridWidget: The ButtonGrid widget is reponsible for displaying the grid of buttons that correspond to questions in the game as well as the labels for each category.
  • Game: This is the Game Pyro object.
  • GameServer: GameServer is the class responsible for spawning Pyro threads of the Game object.
  • Gui: Defines the base Gui class from which the two versions, ServerGui and PlayerGui are derived.
  • LoginDialog: Small dialog the player sees when starting the PlayerGui.
  • Player: This is the Player Pyro object.
  • PlayerAdminDialog: AdminDialog (PlayerAdminDialog) is the dialog used to view connecting players and start the game.
  • PlayerGui: PlayerGui is the main player interface class.
  • PlayerServer: PlayerServer interfaces between the PlayerGui and the remotely accessible Player class.
  • PlayerTableWidget: PlayerTable class contains the table displayed at various times throughout Jeopy.
  • PlotRenderer: Small helper thread to render the final plot using Matplotlib.
  • QuestionDisplayWidget: The Question Display Widget.
  • RuleLoader: Nasty module that valides .jeop files.
  • Server: Baseclass for a Pyro server.
